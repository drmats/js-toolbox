    (inner) lazyNullishCoalesce(left, right)

    Lazy nullish coalescing (inspired by ?? operator, but with right operand eager evaluation prevented using function abstracion).

    Returns its right operand when its left operand is null or undefined, and otherwise returns its left operand.

    In assignments works almost as || operator, but also properly handle values coercing to false (0 or empty string).
