    Time units represented in milliseconds.

    • week - 7 days
    • month - [average month]: 30.4375 days (365.25 days / 12)

    (inner) devEnv(strictopt) → {Boolean}

    Determine runtime environment (is it development or not?). devEnv() -> true/false

    When strict is not set to true then "development environment" can be simulated by storing value of any type under "dev" key in browser's sessionStorage, e.g. sessionStorage[dev] = true.
