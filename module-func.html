    (inner) partial(f) → {function}

    Partial application.

    Bind init arguments to function f and construct a function of smaller arity which accept rest of the arguments.

    Example:

    let f = (a, b) => a + b
f(3, 4)  ->  7
let g = partial(f)(3)
g(4)  ->  7
